How to Reset Password?

Log out and click the Reset Password button on the login page. Enter the email address and password again on the Reset Password page.

How to set the starting time in schedule?

Click the date, click the Add event button, and enter the time in the start time column.

How to add to-do items?

Go to the Home page and click a day to add events

How to delete Event reminder?

Go to the Home page, click the event that has been set for a certain day, find the delete option, and click Delete.

Why is the events I set not being saved?

You have to login successfully, then you can set to-do items and save them.
If you do not have an account, please register first.
